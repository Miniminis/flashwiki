# sudo apt-get install / uninstall

## 1. java&#x20;

```sh
//install 
$ sudo apt update
$ sudo apt install default-jre
$ sudo apt install default-jdk

//uninstall 
$ sudo apt-get autoremove default-jdk openjdk-11-jre
```
