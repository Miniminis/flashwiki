# IP 주소 - 이진수 이해하기

## 1. 개념 이해하기

* 십진법는 말그대로 0-9까지의 수를 이용해서 숫자를 표현하는 방법을 말한다.&#x20;
* 이진법은 0-1까지의 수를 이용해서 숫자를 표현한다.&#x20;

&#x20;

## 2. 연습하기&#x20;

* 181 = 10110101
* 11001010 = 2+8+64+128 = 202&#x20;
* 01010101 = 1+4+16+64 = 85&#x20;
